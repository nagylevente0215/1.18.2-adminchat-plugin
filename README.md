# Minecraft 1.18.2 HelpOp&AdminChat Plugin

> ### **Bukkit 1.18.2 plugin**
> **Usage:** `/ac <message>`
> 
> **Aliases:**
> ```
> - /ac
> - /adminchat
> - /ho
> - /helpop
> ```

The message is **visible** for the **sender** and for **players who has *ac*** permission.

Commands:
> `/ac help` **»** Show the help menu.
> 
> `/ac usage` **»** Show the usage.
